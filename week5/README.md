ตั้งค่า express.json() กับ express.urlencoded()
ไว้สำหรับรับข้อมูลที่ส่งเข้ามา โดยเฉพาะข้อมูลแบบ JSON จะได้เอาไปใช้ใน req.body ได้

สร้างตัวแปร students เป็น array
ไว้เก็บข้อมูลนักศึกษา ข้อมูลจะอยู่ในนี้ตลอดตอนที่เซิร์ฟเวอร์ยังรันอยู่

ใช้ app.listen(3000) เพื่อเปิดเซิร์ฟเวอร์
พอรันไฟล์แล้ว มันจะรอฟัง request ที่พอร์ต 3000

สร้าง route สำหรับดูข้อมูล

GET /api/students
เอาไว้ดูข้อมูลทั้งหมดใน array แล้วส่งกลับไป

GET /api/students/:id
เอา id จาก URL ไปค้นหาใน array
ถ้าเจอก็ส่งข้อมูลกลับ
ถ้าไม่เจอก็ส่ง 404

เพิ่ม route สำหรับจัดการข้อมูล

POST /api/students
รับข้อมูลจาก body
เช็คก่อนด้วย validateStudent
ถ้าถูกก็สร้าง id ใหม่แล้วเพิ่มเข้า array แล้วส่ง 201
ถ้าไม่ถูกก็ส่ง 400

PUT /api/students/:id
เอา id ไปหาใน array
ถ้าเจอก็แก้ข้อมูลตามที่ส่งมา
ถ้าไม่เจอก็ส่ง 404

DELETE /api/students/:id
หา id ใน array
ถ้าเจอก็ลบออก
ถ้าไม่เจอก็ส่ง 404
