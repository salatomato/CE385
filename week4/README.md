สร้าง function ที่รับค่า 3 ตัว คือประเภทของการคำนวณ และ ตัวเลข 2 ตัวโดยใช้เงื่อนไข if-else เพื่อตรวจสอบประเภทการคำนวณ แล้วคืนค่าผลลัพธ์ หากไม่ถูกต้องจะแสดงข้อความแจ้งเตือน

workshop4_2

ส่วนแรกใช้ Callback คือเมื่อข้อมูลพร้อมแล้วจะเรียกฟังก์ชันที่ส่งเข้ามาเพื่อส่งผลลัพธ์กลับไป และตรวจสอบว่ามี error หรือไม่
ส่วนที่สองใช้ Promise โดยถ้าสำเร็จจะส่งค่าด้วย resolve แล้วรับค่าด้วย .then() แต่ถ้าเกิดข้อผิดพลาดจะส่งด้วย reject และจัดการผ่าน .catch() ซึ่งทำให้โค้ดดูเป็นระเบียบและเข้าใจง่ายกว่า

workshop4_3

สร้างฟังก์ชัน simulateAsyncOperation เพื่อจำลองงานที่ต้องรอเวลา โดยรับค่า timeout และคืนค่าเป็น Promise ใช้ setTimeout หน่วงเวลาตามที่กำหนด ถ้าเวลามากกว่า 1000 ms จะส่งข้อมูลกลับว่า “สำเร็จ” แต่ถ้าน้อยกว่าหรือเท่ากับ 1000 ms จะส่งข้อความว่า “Failed”

จากนั้นสร้างฟังก์ชัน performAsyncTask แบบ async ใช้ await รอผลลัพธ์ ถ้าสำเร็จจะแสดง Success พร้อมข้อมูล แต่ถ้าผิดพลาดจะแสดง Error

workshop4_4

สร้าง function 3 ตัวเพื่อดึงข้อมูลจาก Server 1,2 และ 3 แต่ละ Server คือค่าเป็น Promise ใช้ setTimeout เพื่อทำให้โปรแกรมรอก่อนตามเวลาที่ตั้งไว้ แล้วค่อยส่งผลลัพธ์กลับมา

กรณีที่ 1: Promise.any()
จะรอจนกว่าจะมี Promise ตัวใดตัวหนึ่ง “สำเร็จ” ไม่สนใจตัวที่ error ในโค้ดนี้ Server 1 จะสำเร็จก่อน Server 3 จึงแสดงผลข้อมูลจาก Server 1 ถ้าทุกตัว error ถึงจะเข้า .catch()

กรณีที่ 2: Promise.allSettled()
จะรอให้ทุก Promise ทำงานเสร็จ ไม่ว่าจะสำเร็จหรือ error จากนั้นจะส่งผลลัพธ์ทั้งหมดกลับมา เราจะเห็นสถานะของแต่ละ Server ว่าสำเร็จหรือไม่สำเร็จ
